## Overview

Roblox Executor is a Windows-based tool for loading and running custom Lua and Luau scripts in Roblox. It merges execution utilities with a built-in hub, so scripts can be browsed and managed from a straightforward desktop app instead of a more complicated setup.

The app is geared toward quick day-to-day use. You can drag and drop .lua and .txt files, review execution logs, and inspect error output while you work. Multi-instance Roblox support is included as well, making it a practical local workflow for users who want both a script executor and a script hub in one place.

## FAQ

**Is this a Windows-only app?**  
Yes. The project is built for Windows 10 and Windows 11.

**Will older Roblox updates affect script execution?**  
Updates can change how scripts behave, so compatibility may vary over time.

**How is this different from a regular script hub?**  
It brings together a local Windows application, script execution, drag-and-drop loading, logs, and multi-instance support in a single interface.

**Does using a script executor affect account risk?**  
Any third-party script tool can carry account-related risk depending on how it is used and on Roblox's current policies.

**Where are scripts stored?**  
Scripts are handled locally in the app folder and related project directories unless you place them elsewhere.
